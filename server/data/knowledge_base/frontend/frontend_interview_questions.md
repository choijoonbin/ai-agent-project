프론트엔드 개발자 면접 평가 질문 예시 (별도 모음)
1. Web Front-End 기술 역량 관련 질문
* React/Vue 등 선택한 프레임워크의 핵심 개념을 설명해주세요.
* 상태 관리(State management)를 어떤 방식으로 구성했는지 예시 프로젝트 기반으로 설명해주세요.
* CSR/SSR/SSG의 차이를 설명하고 각각 적합한 상황은 무엇인가요?
* 브라우저 렌더링 과정(Reflow/Repaint 포함)을 설명해 주세요.
* REST API 또는 GraphQL 연동 경험을 기술적으로 상세하게 말해주세요.
2. UI/UX 구현 능력 관련 질문
* 사용성을 개선한 경험이 있다면 구체적인 Before/After 사례를 설명해주세요.
* 반응형 웹을 구현할 때 어떤 방식(CSS Grid/Flex/Media Query 등)을 사용했나요?
* 디자인 시안과 실제 개발 간 차이를 조정한 경험이 있나요?
* 접근성(Accessibility) 향상을 위해 어떤 요소를 고려했습니까?
3. 성능 최적화 및 품질 관련 질문
* 성능 저하 문제를 해결한 사례를 기술적으로 설명해주세요.
* 번들 용량 최적화 또는 로딩 속도 개선을 위해 어떤 방법(Lazy Loading, Code Split 등)을 사용했나요?
* Lighthouse 점수를 개선한 경험이 있나요?
* 오류 발생 시 디버깅 과정과 사용한 도구를 설명해주세요.
4. 협업 및 커뮤니케이션 관련 질문
* 백엔드 API 스펙이 변경되었을 때 어떻게 대응하고 조율했습니까?
* 코드 리뷰에서 받은 피드백 중 인상 깊었던 사례는 무엇입니까?
* PM/디자인/백엔드와 가장 의견 충돌이 컸던 사례와 해결 방식을 말해주세요.
5. 문서화, 테스트 및 코드 품질 관련 질문
* 테스트 자동화 도입 경험이 있다면 목적과 효과를 설명해주세요.
* 유지보수성과 확장성을 고려하여 컴포넌트 구조를 설계한 경험이 있다면 말해주세요.
* 기술 문서 또는 개발 가이드를 작성했던 경험을 소개해주세요.

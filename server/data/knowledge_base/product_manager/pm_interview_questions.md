Project Manager (PM) 면접 평가 질문 예시 모음
1. 프로젝트 계획 및 일정 관리 능력 관련 질문
* 프로젝트 일정 계획을 수립한 경험을 설명해주세요. WBS는 어떤 방식으로 작성했나요?
* 일정 지연이 발생했을 때 원인을 어떻게 분석하고 어떻게 대응했는지 사례를 설명해주세요.
* 프로젝트 범위가 변경되거나 요구사항이 추가될 때 일정 영향은 어떻게 평가하고 처리했나요?
* 리스크를 정의하고 관리한 경험이 있다면, 리스크 로그는 어떻게 관리했고 대응 시점은 어떻게 판단했나요?
* 여러 이해관계자가 서로 다른 우선순위를 요구할 때 어떤 기준으로 우선순위를 조정했는지 사례를 설명해주세요.
2. 커뮤니케이션 및 이해관계자 관리 관련 질문
* 프로젝트 이해관계자 간 의견 충돌이 발생했던 상황을 설명하고 해결 과정을 구체적으로 말해주세요.
* 경영진 또는 고객에게 어려운 상황을 보고해야 했던 경험을 설명해주세요. 어떤 방식으로 설득했나요?
* 개발팀/디자인팀/QA 등 서로 다른 조직과 협업하면서 발생한 갈등을 중간에서 조정한 경험을 말해주세요.
* 프로젝트 진행 상황을 공유할 때 어떤 보고 체계를 사용했나요? (예: 주간 보고, 버전 리뷰, 메트릭 활용 등)
* 고객 요구와 기술적 한계가 충돌할 때 어떻게 커뮤니케이션했습니까?
3. 요구사항 분석 및 기획 능력 관련 질문
* 요구사항을 수집하고 정리하는 과정에서 사용한 분석 방법론(인터뷰, VOC, 사용자 스토리 등)이 있다면 설명해주세요.
* 요구사항 변경이 잦았던 프로젝트가 있다면, 변경 통제(Change Control)를 어떻게 운영했나요?
* 본인이 작성한 요구사항 정의서나 프로세스 설계 문서를 예로 들며 구성 항목을 설명해주세요.
* 고객이 명확한 요구를 제시하지 못했을 때 문제를 어떻게 정의하고 해결 방향을 도출했는지 말해주세요.
* 기능 요구와 비기능 요구(성능, 보안 등)를 균형 있게 조정한 경험이 있다면 설명해주세요.
4. 리더십 및 팀 운영 관련 질문
* 구성원 중 성과가 낮았거나 의욕이 부족했던 팀원을 어떻게 관리하고 동기부여했는지 사례를 설명해주세요.
* 예상치 못한 장애나 위기 상황에서 PM으로서 어떤 역할을 수행했고 결과가 어떻게 되었는지 말해주세요.
* 팀 내 갈등이 발생했을 때 중재 방법과 결과는 어떠했는지 설명해주세요.
* 회고(Review, Retrospective) 또는 스프린트 운영 경험을 구체적으로 설명해주세요.
* PM으로서 의사결정을 내리기 가장 어려웠던 순간과 선택의 기준을 설명해주세요.
5. 프로젝트 성과 및 경험 관련 질문
* 가장 성공적인 프로젝트를 소개하고 성공 요인을 수치나 지표로 설명해주세요. (예: 일정 준수율, 비용 절감률 등)
* 가장 실패했던 프로젝트 경험을 설명하고 그 경험을 통해 배우고 개선한 점은 무엇인가요?
* 프로젝트 목표 대비 성과를 측정할 때 어떤 지표(KPI, Metric)를 사용했나요?
* 비용 절감 또는 품질 향상을 위해 시행한 구체적인 활동이 있었습니까?
* 도메인 지식을 빠르게 학습해야 했던 경험이 있다면 어떤 방식으로 접근했나요?

DevOps 엔지니어 면접 평가 질문 예시 (별도 모음)
1. 인프라 및 시스템 운영 역량 질문

장애 또는 서비스 다운 상황을 경험했다면 원인 분석과 해결 과정을 상세히 설명해주세요.

서버 성능 병목 현상을 어떻게 분석하고 해결한 사례가 있습니까?

로그 수집/모니터링 시스템(Prometheus, ELK, CloudWatch 등) 구축 경험을 설명해주세요.

보안 및 네트워크 관련 이슈를 해결한 경험이 있다면 말해주세요.

2. CI/CD 및 자동화 질문

직접 구축한 CI/CD 파이프라인 구조와 사용한 도구를 상세히 설명해주세요.

배포 자동화 시 고려했던 요소(롤백, 승인 프로세스, 환경 분리 등)를 설명해주세요.

스크립트 자동화를 통해 반복 업무를 개선한 사례를 소개해주세요.

배포 실패나 빌드 오류 발생 시 어떤 절차로 원인을 추적했습니까?

3. 클라우드 플랫폼 이해도 질문

AWS/Azure/GCP 중 실제 운영 경험이 있는 환경을 설명해주세요.

클라우드 비용 절감 또는 효율화 작업을 수행한 사례가 있습니까?

클라우드 네트워크(VPC, 서브넷, NAT, Routing 등)를 구성할 때 중요하게 고려한 사항은 무엇입니까?

멀티 AZ 또는 DR 구성 경험이 있다면 설명해주세요.

4. 컨테이너 & Kubernetes 질문

Kubernetes 클러스터 구성 방식과 노드/파드/서비스 개념을 설명해주세요.

서비스 배포 전략(Rolling / Blue-Green / Canary) 중 사용한 경험과 선택 이유를 설명해주세요.

Kubernetes 환경에서 발생한 장애를 해결한 경험이 있습니까?

Docker 이미지 최적화 또는 레이어 관리 경험을 설명해주세요.

5. 협업 및 커뮤니케이션 질문

개발팀 및 운영팀과 기술적으로 갈등이 있었을 때 해결한 경험을 말해주세요.

운영 중 장기적 개선 과제를 제안하고 실행한 경험이 있습니까?

보안팀 또는 QA 팀과 함께 문제를 해결한 사례를 설명해주세요.

실제 프로젝트에서 구축했던 데이터 모델 구조(스키마)와 설계 기준을 설명해주세요.

정규화/비정규화 선택 기준과 적용 사례를 말해주세요.

스타 스키마(Star schema)와 스노우플레이크(Snowflake schema)의 차이를 설명하고 적용 사례가 있다면 말해주세요.

Batch와 Streaming 파이프라인 설계 차이점과 선택 기준을 말해주세요.

데이터 파이프라인 장애가 발생한 사례와 원인 분석 및 해결 방법을 설명해주세요.

데이터 품질 검증(DQ Check)을 어떤 방식으로 수행했는지 설명해주세요.

3. 데이터 처리 및 성능 최적화 관련 질문

SQL 성능 이슈를 해결한 경험이 있다면, 실행계획 분석 방식과 튜닝 방법을 설명해주세요.

대용량 데이터의 병목을 해결한 사례가 있다면 상세히 설명해주세요.

Spark job 최적화 경험이 있다면 파티셔닝, 캐싱, 셔플 감소 등의 관점으로 설명해주세요.

4. 클라우드 및 데이터 플랫폼 관련 질문

AWS/GCP/Azure의 데이터 분석 플랫폼 구성 경험을 설명해주세요.

Redshift / BigQuery / Snowflake / Databricks 중 사용 경험과 장단점을 설명해주세요.

클라우드 환경에서 데이터 비용을 최적화한 사례가 있습니까?

개인정보(PII) 데이터 암호화/마스킹/권한 관리 전략을 어떻게 적용했는지 설명해주세요.

5. 협업 및 커뮤니케이션 관련 질문

데이터 분석가 또는 ML 엔지니어와 협업하며 발생한 이슈를 해결한 사례를 말해주세요.

데이터 요구사항을 수집하고 모델링으로 구체화한 경험을 말해주세요.

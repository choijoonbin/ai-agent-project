QA 면접 평가 질문 예시 모음 (별도 정리)
1. 테스트 전략 및 설계 능력 질문

테스트 케이스를 설계할 때 어떤 기준으로 테스트 범위와 우선순위를 결정하나요?

요구사항이 명확하지 않거나 불완전한 경우 어떻게 테스트 시나리오를 구성하나요?

가장 복잡했던 테스트 케이스 설계 경험을 설명해주세요.

비기능 테스트(성능/보안/부하/내구성 등) 수행 경험이 있다면 설명해주세요.

2. 테스트 자동화 및 도구 관련 질문

자동화 테스트 구축 경험을 설명하고, 어떤 테스트 범위를 자동화했는지 말했다 주세요.

Selenium, Cypress, Playwright, JMeter 등 특정 도구를 선택한 이유는 무엇인가요?

자동화된 테스트가 실패했을 때 어떤 방식으로 원인을 분석하고 대응했습니까?

CI/CD 파이프라인에서 테스트 자동화를 적용한 경험을 설명해주세요.

3. 버그 분석 및 문제 해결 능력 질문

가장 어려운 장애 또는 버그를 해결한 경험을 설명해주세요.

Severity와 Priority 차이를 설명하고 실제 판단 기준을 사례와 함께 말해주세요.

문제의 원인을 정확히 찾기 어려웠던 상황에서 어떤 방식으로 접근했나요?

로그/모니터링/APM 도구를 활용한 분석 경험이 있으면 설명해주세요.

4. 품질 관리 및 운영 관련 질문

릴리즈 품질을 보장하기 위해 어떤 품질 지표를 관리했나요?

배포 이후 운영 중 장애가 발생했을 때 어떤 절차로 대응했는지 설명해주세요.

테스트 완료 후 품질 보고서를 작성할 때 핵심적으로 포함해야 할 내용은 무엇인가요?

5. 협업 및 커뮤니케이션 질문

개발자나 기획자와 기능 오류 여부를 두고 의견 충돌이 있었던 경험과 해결 방법을 말해주세요.

버그 보고서를 작성할 때 가장 중요하게 생각하는 요소는 무엇인가요?

문제를 근거 기반으로 설득해야 했던 경험을 설명해주세요.

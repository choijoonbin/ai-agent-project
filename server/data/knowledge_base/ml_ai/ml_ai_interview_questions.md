ML / AI 엔지니어 면접 평가 질문 예시 (별도 모음)
1. ML/DL 기술 역량 관련 질문

다뤄본 머신러닝 알고리즘 중 가장 자신 있는 모델을 설명하고, 어떤 문제에 적용했는지 사례를 말해주세요.

모델의 과적합을 방지하기 위한 방법을 설명해주세요.

데이터 불균형(Imbalanced Data) 문제를 해결한 경험이 있으면 설명해주세요.

Transformer 구조의 핵심 개념과 기존 RNN 대비 장점을 설명해주세요.

모델 성능 평가 지표(Accuracy, F1, ROC-AUC 등)를 어떤 기준으로 선택했는지 설명해주세요.

2. 데이터 처리 및 파이프라인 관련 질문

데이터 전처리 단계에서 가장 어려웠던 문제와 해결 과정을 사례로 설명해주세요.

Feature Engineering 과정에서 의사결정 기준은 무엇이었습니까?

Streaming / Batch 파이프라인 구축 경험이 있다면 구성과 사용 도구를 설명해주세요.

MLflow 또는 실험 관리 도구 사용 경험을 구체적으로 설명해주세요.

3. 모델 배포 및 운영(MLOps) 질문

모델 서빙(FastAPI, Triton 등) 경험을 설명하고 구성 방식을 관련 아키텍처와 함께 설명해주세요.

모델 성능 저하(Drift) 문제를 어떻게 탐지하고 대응했는지 사례를 설명해주세요.

자동 재학습 또는 파이프라인 구축 경험이 있다면 설명해주세요.

GPU 선택 기준 또는 비용 최적화 방법을 말해주세요.

4. 논문 분석 및 연구 능력 질문

최근 읽은 AI 논문 중 가장 인상 깊었던 연구와 적용 가능성을 설명해주세요.

오픈소스 모델을 커스터마이징하거나 개선한 경험이 있다면 말해주세요.

벤치마킹 실험을 설계할 때 고려했던 요소는 무엇인가요?

5. 협업 및 문제 정의 능력 질문

애매한 비즈니스 요구를 ML 문제로 재정의한 사례를 설명해주세요.

모델 성능 개선 요구와 실제 서비스 운영 제한(비용, 속도) 간 트레이드오프 조정 경험이 있습니까?

데이터 분석가/서비스 기획/백엔드 개발자와의 협업 과정에서 겪은 어려움과 해결 방법을 말해주세요.
